#  Техническое задание

## Цель проекта

Разработать демонстрационное приложение на языке **C++** с использованием криптографической библиотеки **Crypto++**, реализующее основные алгоритмы защиты информации.



## Задачи

1. Изучить библиотеку Crypto++ и её архитектуру (pipeline / filter chains)
2. Реализовать хеширование данных алгоритмами SHA-256, SHA-1, MD5
3. Реализовать аутентификацию сообщений через HMAC-SHA256
4. Реализовать симметричное шифрование AES-128 в режиме CBC
5. Реализовать деривацию ключа из пароля через PBKDF2-HMAC-SHA256
6. Обеспечить корректную обработку ошибок и безопасное хранение ключей
7. Оформить проект с документацией и Wiki



## Требования к реализации

### Функциональные требования

| № | Требование |
|---|---|
| 1 | Программа вычисляет SHA-256, SHA-1, MD5 от произвольной строки |
| 2 | Программа вычисляет HMAC-SHA256 с заданным ключом |
| 3 | Программа шифрует и расшифровывает строку алгоритмом AES-128 CBC |
| 4 | Программа поддерживает шифрование с паролем через PBKDF2 |
| 5 | Ключи и IV генерируются криптографически стойким ГПСЧ |
| 6 | Результаты выводятся в HEX и Base64 |

### Нефункциональные требования

- Язык: **C++17**
- Библиотека: **Crypto++ 8.x**
- Сборка: **CMake 3.10+** и **Visual Studio 2019/2022**
- Ключи хранятся в `SecByteBlock` (автоматическое обнуление памяти)
- Все криптографические операции обёрнуты в `try-catch`



## Ограничения

- MD5 и SHA-1 реализованы **только в образовательных целях**
- Проект является **демонстрационным** и не предназначен для продакшн-использования
